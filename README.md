# TryHackMe Lab Writeups

This repository contains my TryHackMe lab writeups as I build hands-on cybersecurity fundamentals, with a focus on web security, enumeration, and vulnerability reasoning.

The goal of these writeups is not just to document flags, but to explain the thought process behind each room: what I noticed, what I tested, what worked, what failed, and what I learned.

## Focus Areas

- Web application security
- Enumeration methodology
- Linux and networking basics
- Vulnerability identification
- Beginner CTF methodology
- Clear technical writing

| Room | Category | Key Concepts |
|---|---|---|
| Neighbor | Web / IDOR | Source review, guest login, authorization bypass |

## Notes

These write-ups are for learning and documentation purposes. I avoid posting active flags when possible and focus on methodology, vulnerability concepts, and remediation.
